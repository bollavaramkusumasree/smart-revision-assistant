# 📚 Smart Revision Assistant

A Streamlit-based web application that converts Wikipedia topics into structured revision notes for students.

---

## 🚀 What it does

- Takes any academic topic as input
- Fetches content from Wikipedia
- Converts it into clean revision format:
  - Short Summary
  - Key Points
  - Important Concepts

---

## 💡 Why this project?

Students often struggle with:
- Long textbook content
- Unstructured notes
- Last-minute revision

This tool solves that by **compressing knowledge into revision-friendly format**.

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Wikipedia API (requests)

---

## 📸 Features

- Simple UI (web app)
- Fast topic search
- Structured learning output
- No login required

---
